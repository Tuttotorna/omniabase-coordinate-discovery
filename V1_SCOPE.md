# OMNIABASE V1 - Scope, Boundaries, and Allowed Claims

## What Omniabase V1 is

Omniabase V1 is an experimental multibase structural analysis framework.

It does not read semantic meaning.
It does not infer intent.
It does not predict by narrative interpretation.

It measures how a signal behaves when the same state is represented across multiple numerical bases.

The core intuition is:

**a system may look flat or random in one representation while still carrying structural fingerprints across many representations.**

Omniabase V1 extracts those fingerprints and uses them as diagnostic signals.

---

## Core principle

For a given scalar or derived observable:

1. represent the value across bases 2 through 16
2. extract simple structural signatures
3. track how those signatures behave across time, regimes, couplings, projections, or generators

The framework currently uses simple non-semantic signature families such as:

- digit sum span
- repetition span
- transition diversity / transition entropy-style score
- means and dispersions of these quantities
- relative deltas over time or over parameter scans

---

## What Omniabase V1 is NOT

Omniabase V1 is not:

- a universal theory of truth
- a replacement for dynamical systems theory
- a proof engine
- a semantic model
- a universal entropy oracle
- a guaranteed detector of hidden variables
- a direct alpha-generation engine for real markets
- a final scientific theory of chaos

Any claim stronger than the benchmark evidence is outside V1 scope.

## What V1 has actually demonstrated

### 1. Regime sensitivity in canonical dynamical systems

Omniabase V1 showed initial evidence of distinguishing structural regimes in:

- logistic map
- Hénon map
- coupled map lattice
- Lorenz system
- 4D hyper-Lorenz-style benchmark

This includes:
- periodic vs chaotic separation
- transition amplification
- ordered-window detection
- multidimensional scaling

### 2. Separation between order and event

In several benchmarks, a preliminary coordinate pair was constructed:

- `order_score`
- `event_score`

This pair showed initial evidence of separating:

- structural calm / order
- transition tension / instability

This separation is benchmark-supported, not yet universal.

### 3. Real-time alert behavior

On Lorenz regime-switch simulations, rolling order/event scores produced early alerts after a control-parameter switch.

This supports:

- fast structural regime-shift detection
- rolling sentinel-style monitoring

This is a tested operational pattern in benchmark conditions.

### 4. Noise robustness

In tested scenarios, moderate sensor noise increased baseline turbulence but did not destroy the main structural signal.

This supports:

- initial noise robustness
- resistance to false alarms in tested conditions

This is benchmark-specific evidence, not a universal robustness theorem.

### 5. Synchronization measurement

On coupled Lorenz systems, Omniabase measured synchronization through the multibase structure of the error signal.

This supports:

- relational structural sensing
- synchronization scoring without direct semantic interpretation

### 6. Hidden-dimension sensitivity in controlled projection tests

In a controlled 4D-to-2D benchmark, the visible projection of the full system differed strongly from a naive 2D model.

This supports:

- initial sensitivity to latent-variable influence in projection benchmarks

This is not yet full hidden-state inference.

### 7. Generator-type discrimination

Omniabase distinguished, in benchmark conditions, between:

- deterministic chaotic output
- simple linear PRNG output
- stronger PRNG-like output

This supports:
- generator-type discrimination by structural footprint

### 8. Structured signal analysis beyond textbook dynamical systems

On synthetic market-like signals, Omniabase distinguished:

- pure noise
- trend plus noise
- mean reversion
- latent chaotic driver

This supports:
- extension from canonical dynamics to structured synthetic signals

The following claims are allowed because they are supported by current experiments:

3. It shows initial evidence of distinguishing different dynamical regimes.
